SM PBL – Linear Congruential Problem

Introduction

The Linear Congruential Generator (LCG) is one of the oldest and simplest methods for generating pseudo-random numbers using a recurrence relation.

Formula: Xi+1 = (aXi + c) mod m

Problem Statement

Generate a sequence of random numbers using the LCG method with parameters:

a = 5, c = 3, m = 16, X₀ = 7

Steps to Solve

  1. Start with the seed X₀ = 7
  2. Use the formula: Xi+1 = (aXi + c) mod m
  3. Repeat until sequence repeats

Calculations

iXiRi = Xi/m

Validation (Hull–Dobell Theorem)

  • gcd(c, m) = gcd(3,16) = 1 ✅
  • (a−1) = 4 divisible by all prime factors of m (2) ✅
  • m = 16 is multiple of 4 → (a−1)=4 is multiple of 4 ✅

Hence, the generator achieves a full period of m = 16 ✅

Conclusion

The LCG method successfully generated a full-period pseudo-random sequence. This implementation validates theoretical conditions using the Hull–Dobell theorem.
